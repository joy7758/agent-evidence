# Environment

- Required Python: 3.11 or newer.
- Observed repair environment: macOS with Python 3.14.5.
- Installation command: `python -m pip install -e .`
- Verification tools: `pytest`, `ruff`, and `jsonschema`.
- Private data dependency: none.
- Clinical/customer data dependency: none.
- Network dependency after package download: none for the bundled fixtures.

The release candidate is intended to run locally from a clean checkout or extracted source package after Python dependencies are installed.

## Abstract / 摘要

**English**  Execution Evidence Object is a standards specimen for treating AI runtime evidence as a portable and verifiable digital object. The repository includes a named object model, a JSON schema, example objects, integrity verification, cross-framework export prototypes, and an FDO-style mapping layer. It does not claim formal standard adoption. It demonstrates a concrete baseline for discussing AI execution evidence in object-oriented terms.

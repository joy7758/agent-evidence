# Resource abstract draft

We introduce an open, validator-backed framework for portable execution evidence in scientific workflows. The framework packages an executed operation, policy, inputs, outputs, provenance links and validation outcomes into a small evidence pack consisting of a bundle, receipt and summary. A command-line validator recomputes digests, checks profile conformance, verifies evidence linkage and returns deterministic pass/fail codes. We release public scientific-workflow and tool-use workflow packs, failure-injection cases, a conformance CI workflow and an independent checker to support offline review. In validation experiments, untampered packs pass under pinned containers, while tampered inputs, outputs, missing policies and broken evidence links fail with stable reason codes. The framework is intended as a reusable reproducibility layer that can complement existing workflow metadata packages rather than replace them.

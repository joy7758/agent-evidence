# Execution Evidence and Operation Accountability Profile

EEOAP means Execution Evidence and Operation Accountability Profile.

EEOAP defines a minimal operation-accountability profile for completed
operations. It gives agents a bounded way to package operation, policy,
evidence, provenance, and validation information into a reviewable evidence
statement.

This profile is useful for agents because it gives them a machine-checkable way
to package a completed operation into a reviewable evidence statement.

## Core surfaces

- Operation accountability statement
- Profile-aware validator
- Evidence object
- Validation report
- Controlled examples
- Local demo and tests

## Claim scope

EEOAP is a local profile-aware validation surface for completed operations.

It is not legal compliance.

It is not external certification.

It is not a full FDO implementation.

It is not a general provenance standard.

It is not deployment proof.

It is not publication evidence.

Use `docs/CLAIM_SCOPE.md` for the repository-wide claim boundary.

# AGT upstream contribution copy pack

This draft collects reusable public wording for the merged Microsoft Agent Governance Toolkit documentation-only interoperability contribution.

## 1. Safe factual core

- Merged a docs-only interoperability note into Microsoft Agent Governance Toolkit documentation.
- Documented a path to map AGT runtime evidence into an external operation-accountability profile.
- Framed AGT runtime evidence as upstream evidence material for a downstream accountability statement.
- Advanced the topic into upstream public discussion through a GitHub issue and merged documentation PR.
- Kept the contribution documentation-level: no AGT runtime, policy, identity, sandboxing, package, test, or official evidence-schema changes.
- Made no claim of official EEOAP schema adoption, AGT runtime integration, or replacement of AGT evidence verification.

## 2. Claims to avoid

- Microsoft adopted EEOAP.
- AGT officially supports EEOAP.
- AGT integrated my schema.
- AGT has an official EEOAP runtime integration.
- Microsoft accepted my standard.
- Microsoft endorsed my external validator.
- AGT replaced its evidence verification with EEOAP.
- The PR added runtime support, policy enforcement changes, identity changes, or sandboxing changes.
- The PR changed AGT's official evidence schema.
- The prototype is an official AGT evidence format.

# Public Recruitment Call: External AEP-Media Reproducibility Reviewers

Date: 2026-06-01

AEP-Media is an open-source Python package for local validation of
time-aware, mobile-video-style media evidence bundles. It provides command-line
validators, fixture examples, bundle build and verification commands, and
small synthetic mobile-video-style fixtures for reproducibility review.

The current need is external reproducibility review. I am looking for one or
two reviewers to start from a fresh clone, follow the README and mobile-video
walkthrough, run the listed CLI commands and targeted tests, and report whether
the workflow is understandable and reproducible.

The expected deliverable is one actionable GitHub issue or pull request with
environment details, exact commands, output summaries, failures, unclear
documentation, and concrete suggestions. Negative feedback is welcome.

Payment, if agreed, compensates time spent testing and reporting. It does not
buy praise, stars, endorsement, citations, authorship, positive conclusions, or
publication outcomes.

Reviewers must use only the repository's small synthetic fixtures. Do not
upload private media, confidential logs, real evidence files, or large binaries.

Claim boundary: AEP-Media supports local validation and fixture-based adapter
ingestion. It does not claim legal admissibility, chain of custody, real PTP
proof, real C2PA verification, or production deployment.

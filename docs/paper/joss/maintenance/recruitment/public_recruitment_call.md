# Call for external reproducibility reviewers

## Who should respond

Python CLI users, open-source documentation contributors, reproducibility
testers, and researchers interested in media evidence or provenance workflows.

## Task

Start from a fresh clone of `https://github.com/joy7758/agent-evidence` and run
the AEP-Media mobile-video-style walkthrough. The task uses only small
synthetic fixtures already in the repository.

## Deliverable

Post one actionable GitHub issue comment or PR with operating system, Python
version, shell, exact commands, output summary, first failure if any, unclear
documentation, and suggested fixes.

## Boundaries

AEP-Media supports local validation and fixture-based adapter ingestion. It does
not claim legal admissibility, chain of custody, real PTP proof, real C2PA
verification, production deployment, or proof that the underlying event is
truthful.

Do not upload private media, confidential logs, real evidence files, or large
binaries.

## Payment disclosure

Payment, if agreed, compensates time spent testing and reporting. It does not
buy praise, stars, endorsement, citations, authorship, positive conclusions, or
publication outcomes. Negative feedback is welcome.

# SoftwareX Cover Letter Draft

Dear Editor,

I submit the manuscript "AEP-Media: Reusable Research Software for Offline Validation of Time-Aware Media Evidence Bundles" for consideration by SoftwareX.

AEP-Media is reusable research software for representing a media-bearing operation as a locally checkable evidence bundle. The software provides a media evidence profile, JSON schemas, controlled examples, profile-aware validators, an offline bundle builder and verifier, strict declared time-trace validation, adapter-only ingestion interfaces, demos, tests, and reproducibility-oriented release reports.

The software is intended for researchers studying operation accountability, media evidence packaging, provenance boundaries, and reproducible validation artifacts. Its evaluation package includes valid, invalid, and tamper cases with expected pass/fail behavior and machine-readable failure codes. The fixture-based path is reproducible without LinuxPTP, FFmpeg, ffprobe, or C2PA tools.

The contribution is bounded. AEP-Media supports local validation and fixture-based adapter ingestion. It does not claim legal admissibility, non-repudiation, trusted timestamping, real PTP proof, a full MP4 PRFT parser, real C2PA signature verification, chain of custody, or production deployment.

The repository is publicly available at https://github.com/joy7758/agent-evidence and is licensed under Apache-2.0. A final release archive and AEP-Media-specific DOI should be prepared before submission if required by the journal workflow.

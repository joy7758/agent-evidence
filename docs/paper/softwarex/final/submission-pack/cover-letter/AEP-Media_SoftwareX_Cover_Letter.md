# AEP-Media SoftwareX Cover Letter

Dear Editor,

I submit "AEP-Media: Reusable Research Software for Offline Validation of Time-Aware Media Evidence Bundles" for consideration as an Original Software Publication in SoftwareX.

AEP-Media is reusable research software within the `agent-evidence` repository. It provides a media evidence profile, schemas, validators, examples, command-line tools, offline bundle verification, strict declared time-trace validation, adapter-only ingestion, and reproducible evaluation matrices for studying time-aware media evidence bundles and operation accountability.

The software is intended for researchers working on media evidence packaging, provenance boundaries, reproducible validation artifacts, and operation accountability. It is released under Apache-2.0 and includes documentation, tests, examples, evaluation reports, reproducibility commands, and Supplementary file S1. The AEP-Media v0.1.0 software release is available at https://github.com/joy7758/agent-evidence/releases/tag/aep-media-v0.1.0 and archived at Zenodo DOI `10.5281/zenodo.20107097`.

The manuscript preserves explicit claim boundaries. AEP-Media supports local validation and fixture-based adapter ingestion; it does not claim legal admissibility, non-repudiation, trusted timestamping, real PTP proof, full MP4 PRFT parsing, real C2PA signature verification, chain of custody, or production deployment.

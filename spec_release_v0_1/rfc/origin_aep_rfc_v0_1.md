# Origin AEP Execution Evidence Specification v0.1

Status: experimental local specification draft.

Origin AEP v0.1 defines a compact local object model for agent execution evidence: event, trace, audit, and local receipt. The term local receipt does not mean external certification.

Adapters may map GitHub PRs, Graphite stacked PRs, Cursor-style agent runs, MCP-style tool calls, and Origin-style execution runtimes into the common event shape. These are compatibility targets, not official integrations or vendor endorsements.

This draft does not claim legal compliance, external certification, standard adoption, publication acceptance, production readiness, or external validation.

Affected clauses: EEOAP-001, EEOAP-002, EEOAP-003, EEOAP-004, EEOAP-005.

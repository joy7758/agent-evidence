# Support

Use GitHub issues for support, bug reports, documentation questions, and feature
requests:

<https://github.com/joy7758/agent-evidence/issues>

## Before Opening an Issue

Please include:

- the command you ran;
- expected behavior;
- actual behavior;
- Python version;
- operating system;
- package version, release tag, or commit hash;
- a minimal synthetic or redacted fixture if the issue involves input data.

Do not post private media, private evidence, credentials, tokens, personal
identifiers, or sensitive logs. If an issue concerns private evidence, describe
the failure using a reduced synthetic fixture.

## Supported Environment

The project currently targets Python 3.11+ and local command-line usage.
Development checks run through pytest and GitHub Actions.

## Scope Boundaries

Support is limited to the repository's documented local validation and evidence
packaging behavior. This project does not provide legal advice, production
forensic certification, chain-of-custody guarantees, trusted timestamping, real
PTP proof, full MP4 PRFT parsing, or real C2PA signature verification.

For security-sensitive reports, prefer a minimal description first and avoid
posting sensitive artifacts publicly.

# Final Submission Overview

## 项目名称

TRPS：面向金融机构交易前风险治理的可审计智能决策与模拟执行平台

## English Project Name

TRPS: An Auditable Intelligent Decision and Simulated Execution Platform for Pre-Trade Risk Governance in Financial Institutions

## 参赛组别

AFAC2026 初创组

## 项目定位

TRPS 是面向交易前风险治理的可审计智能决策与模拟执行平台。它把高风险金融决策请求转成可复核的风险结构、策略闸门、人工复核状态和审计回执。

## 不是什么

- 不是自动交易机器人。
- 不是个人投资建议工具。
- 不是监管认证系统。
- 不证明真实市场盈利。

## 核心闭环

`scenario input -> belief state -> risk distribution -> policy gate -> human review -> simulated execution -> audit receipt`

## 目标客户

- 券商资管
- 财富管理平台
- 银行投顾中台
- 风控与合规团队

## 当前证据

- `synthetic controlled demo`
- `static offline demo`
- `policy gate`
- `audit receipt`
- `validation reports`
- `readiness score`

本包是本地最终报名准备包，供人工复制门户字段、检查 ZIP 内容、演练路演和准备答辩使用。它不是已提交、已批准或已外部评审状态。

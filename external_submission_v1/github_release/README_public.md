# Origin AEP v0.1 Public Review Package

Origin AEP v0.1 is a local experimental protocol-candidate package for agent execution evidence. It binds intent, context, action, evidence, risk, and authority into a deterministic local record.

This package is prepared for human review. It does not create a GitHub release, post a discussion, submit a paper, send email, request endorsement, or contact any external party.

## Quickstart

```bash
python3 external_submission_v1/publish_gate/RELEASE_READINESS_SUMMARY.py
python3 spec_release_v0_1/validator/spec_validator.py spec_release_v0_1/examples/github_pr_example.json

```

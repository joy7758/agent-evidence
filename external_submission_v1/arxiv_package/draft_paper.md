# Origin AEP v0.1: A Local Event Surface for Agent Execution Evidence

## Abstract

Agent-assisted software development increasingly produces changes whose evidence must be reviewed after the original runtime has ended. Origin AEP v0.1 is a local protocol-candidate package that models this evidence through a canonical event surface: intent, context, action, evidence, risk, and authority.

## Related Work Review Requirement

This draft intentionally does not fabricate a literature review. Before any archive submission, a human author should add and verify related work covering software supply-chain attestations, provenance systems, audit logs, pull request automation, agent runtime tracing, and evidence packaging.

## Boundary

Local evidence is not external truth. Origin AEP v0.1 cannot prove remote service state, externally verified identity, legal evidentiary status, security readiness, or production readiness.

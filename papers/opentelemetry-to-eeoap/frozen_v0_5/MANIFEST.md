# Manifest

This manifest lists every file in `frozen_v0_5/` and its role.

| File | Role |
|---|---|
| `README.md` | Package-level overview, commit chain, artifact claim, reproduction command, and freeze boundary. |
| `paper_v0_4.md` | Frozen paper draft used as the v0.5 package paper text. |
| `references_draft.md` | Draft reference entries and citation verification notes. |
| `artifact_freeze_note.md` | v0.4 artifact freeze note copied into the frozen package. |
| `submission_checklist.md` | Paper files, artifact files, reproduction commands, tests, limitations, and missing submission items. |
| `claim_boundary.md` | Positive claim, evidence basis, non-claims, and scope of validator success. |
| `evaluation_summary.md` | Valid trace, invalid trace diagnostics, validator result, and test evidence summary. |
| `reviewer_positioning.md` | Reviewer-facing positioning and objection handling. |
| `FREEZE_STATUS.md` | Internal freeze status and external-submission readiness boundaries. |
| `NEXT_ACTIONS.md` | Prioritized next actions split into internal archive, external submission, and deferred engineering paths. |
| `CHECKSUMS.sha256` | SHA-256 checksums for all files in this directory except `CHECKSUMS.sha256` itself. |

# SoftwareX Author-Confirmed Metadata v1.20

## Purpose

This directory records author-confirmed public metadata for later SoftwareX
preparation of the OpenTelemetry-to-EEOAP package. It resolves the author,
affiliation, corresponding contact, support-contact preference, ORCID, country,
license, keywords, and funding fields that were left open in version 1.19.

## Source

- Source branch: `softwarex-otel-eeoap-release-candidate`

## Scope

This is a planning and documentation step only. It does not update root
metadata, public release metadata, SoftwareX template files, CFF, CodeMeta JSON,
runtime code, tests, fixtures, generated outputs, the EEOAP schema, tags, DOI,
GitHub Release, or formal submission state.

Private phone numbers are excluded from public metadata. No phone number value
is recorded in this directory.

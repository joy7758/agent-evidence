# Unprovable Points Matrix

| representation | from it alone, what remains unprovable | why this matters |
| --- | --- | --- |
| logs-only | policy 是否稳定绑定；refs 是否闭合；evidence block 是否形成；validation path 是否明确 | 无法把运行记录提升为局部验证单元 |
| provenance-only | governing policy；artifact sufficiency；validation status | lineage 不等于 accountability closure |
| policy-only | concrete executed operation；实际输入输出；evidence continuity；validation result | 规则依据不等于执行事实 |
| audit-trail-only | portable local statement 是否成立；离开原系统后能否独立检查；evidence continuity 是否足够 | 审计记录可能很丰富，但仍过大且实现耦合 |
| boundary-based accountability statement | 全局 workflow history；完整组织治理流程；广泛生态互操作已完成 | 这些不属于本文问题边界 |

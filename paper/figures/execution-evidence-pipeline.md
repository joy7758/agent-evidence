# Execution Evidence Pipeline

Keywords: EEOAP, execution evidence, operation accountability, validator, FDO-style mapping, paper_case.

```text
Agent Operation
  -> Runtime Event
  -> Evidence Bundle
  -> Signed Manifest or Integrity Binding
  -> Validator
  -> Verification Receipt
  -> Reviewer Summary
```

中文说明：该流程展示 AI 智能体操作如何转化为可离线复核的执行证据与问责材料。

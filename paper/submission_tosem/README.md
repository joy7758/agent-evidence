# TOSEM Submission Packaging Scaffold

This folder collects the first submission-oriented packaging scaffold built from the stable English draft in `paper/tosem_en/` and the current repository-grounded artifact evidence.

It is not a second manuscript track. Its purpose is to organize what is already written into a TOSEM-facing submission package plan: section mapping, figure and table handling, reference planning, claim checking, and remaining blockers before template conversion.

The source of truth for prose remains the stable English draft and the current repository artifacts. This folder only prepares that material for the next packaging phase.

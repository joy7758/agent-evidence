Dear Editor,

We submit this manuscript as a SoftwareX research software article. It presents
a research software artifact that implements a deterministic transformation
from OpenTelemetry trace data into a repository-scoped structured execution
evidence representation.

The contribution is a fully reproducible, reusable trace-to-evidence software
system including:

- schema definition
- trace conversion pipeline
- validation framework
- baseline comparison
- reproducibility scripts

The work is designed to be executed directly as a software artifact, requiring
no additional infrastructure beyond a standard Python environment, and is
intended for readers who need a reusable transformation boundary between
existing OpenTelemetry traces and validator-readable evidence artifacts.

The manuscript does not claim a new telemetry standard, protocol replacement,
certification, or production deployment.

We believe this submission aligns with SoftwareX scope as a research software
article focused on reproducible computational artifacts. The accompanying software distribution and support material are publicly available for inspection, validation, and reuse at https://github.com/joy7758/agent-evidence/releases/tag/v2.0-softwarex-clean.

# Data Availability Statement

The paper uses repository-contained `paper_case` artifacts under
`examples/paper_case/`. The case is designed for local reproduction and review
of EEOAP (Execution Evidence and Operation Accountability Profile,
执行证据与操作问责配置文件) evidence objects.

The paper does not use human participants. It does not use patient data. It
does not use an external private dataset. The current data and evidence
objects are repository-contained demonstration artifacts for a scoped
validator path.

The reproduction command is:

```bash
make paper-demo
```

Public release and DOI (Digital Object Identifier, 数字对象标识符) availability
are not claimed unless a public release, archive, or DOI is actually created
later. If that state changes, this statement must be updated only after the
public metadata exists and has been manually verified.
